# GHub

App que consulta API do GitHub.

Para visualizar o layout:
[Layout Figma](https://www.figma.com/file/YxgOtQhIitAEyuuw4Dzdoe/GitHub-App?node-id=0%3A1).


## Instalação:
Para instalar:

```bash
yarn install

```

## Rodar a App:

```bash
yarn start

```
